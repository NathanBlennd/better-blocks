# Better Blocks

Registers custom blocks.

## Installation

1. Clone the project to your desired location
2. Open the command line in the project folder and install the packages:

```shell
npm install
```

## Usage

Run the scripts for the included **hero** block
```shell
npm run start:hero
npm run build:hero
```
